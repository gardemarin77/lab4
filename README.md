GGR472 Lab 4 – Toronto Traffic Collisions Hexgrid Map
This map shows a hexgrid map of pedestrian and cyclist road collisions in Toronto between 2006 and 2021. Each hexagon covers a 0.5km area and is coloured by how many collisions happened inside it — pale yellow means few collisions, dark red means many.

Click any hexagon to see the number of collisions in that area
Check/uncheck the "Show collision points" box to toggle the raw collision dots on and off
Use the +/− buttons in the top-right corner to zoom in and out

Data source
City of Toronto Open Data — Motor Vehicle Collisions Involving Killed or Seriously Injured Persons
https://open.toronto.ca/dataset/motor-vehicle-collisions-involving-killed-or-seriously-injured-persons/


Mapbox GL JS — map rendering
Turf.js — GIS analysis (bounding box, hexgrid, spatial join)
